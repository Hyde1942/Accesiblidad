Qué es la Accesibilidad Web?

La Accesibilidad Web es simplmente garantizar que un sitio o servicio web este disponible para el mayor número de personas.

La web esta basada en un principio de universalidad, no importa el hardware, software, lenguaje, cultura, geografía, habilidades físicas o mentales.

Imagenes

Tienen que tener un alt tag descriptivo pero no demasiado largo. En el caso de tener que suministrar una descripción mas detallada utilizar otro metodo, como ARIA para suministrar la descripción.


									<img src="images/path.jpg" alt="Descripcion 250 max" />

Para las images decorativas se tiene que poner un alt tag vacio.


									<img src="images/path.jpg" alt="" role="presentation" />

Contraste y Color

"Contraste" son colores en segmentos opuestos de la rueda del color.

Para cumplir la normativa AA, El contraste minímo de un texto debe ser 4:5

Para cumplir la normativa AAA, El contraste minímo de un texto debe ser 7:5

Font-size & Line Height

El font-size del texto del página debería setearse en un valor entre los 15px - 20px. De acuerdo con la W3C el line-height debería ser 1.5 veces el tamaño del line spacing. También el largo de los textos de no debería de exceder los 85 carácteres. 

								
										.p {
											font-size: 1em;
											max-width: 65ch; /* Limits paragraph width */
											line-height: 1.5;
										}

Flexbox

Es un nuevo módulo en CSS. El cual define un nuevo tipo de elemento con el property display:flex. El objectivo del Flexbox es permitir acomodar items flexibles dentro de un contendor flexible.

También nos permite definir un Orden, para cada uno de los items dentro del contenedor. Lo que nos viene muy bien en el tema de Accesibilidad. Y mantener un orden lógico.
