Que es la Accesibilidad Web?

La Accesibilidad Web es simplmente garantizar que un sitio o servicio web este disponible para el mayor numero de personas.

La web esta basada en un principio de universalidad, no importa el hardware, software, lenguaje, cultura, geografia, habilidades fisicas o mentales.
